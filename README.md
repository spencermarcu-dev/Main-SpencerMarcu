# SpencerMarcu

Personal portfolio website for Spencer Marcu, built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/spencermarcu).

## Overview

A clean, responsive portfolio site featuring:

- **Hero** section with a brief introduction
- **About** section
- **Projects** showcase
- **Contact** section linking to GitHub

## Development

To run locally:

```bash
gem install bundler jekyll
bundle exec jekyll serve
```

Then open <http://localhost:4000> in your browser.

## Deployment

Pushes to the `main` branch are automatically deployed via GitHub Pages.
